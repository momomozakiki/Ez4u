# Tailwind CSS Examples

This directory contains purpose-specific examples for Tailwind CSS patterns and best practices.

## Available Examples

- **[utility-patterns.md](utility-patterns.md)** - Basic utility combinations and state variants
- **[component-patterns.md](component-patterns.md)** - Reusable component class patterns
- **[responsive-patterns.md](responsive-patterns.md)** - Breakpoint and responsive design patterns
- **[token-usage.md](token-usage.md)** - Design token usage and theme-aware patterns

## Usage

Each example file focuses on specific Tailwind patterns:
1. **Utilities** - Common utility combinations and anti-patterns
2. **Components** - Component composition and variant patterns
3. **Responsive** - Mobile-first and breakpoint patterns
4. **Tokens** - Design token usage and theme integration
