# TRAE Skill & Rule Creation Examples

This directory contains purpose-specific examples for creating new skills and rules.

## Available Examples

- **[categorization-examples.md](categorization-examples.md)** - How to decide between creating a rule or a skill.
- **[structure-examples.md](structure-examples.md)** - The mandatory folder and file structure for skills and rules.
- **[frontmatter-patterns.md](frontmatter-patterns.md)** - Correct and incorrect frontmatter for skills and rules.
- **[template-usage.md](template-usage.md)** - How to use the provided templates to create new skills and rules.

## Usage

Each example file focuses on a specific aspect of the creation process:
1. **Categorization** - Deciding whether your guidance is a rule or a skill.
2. **Structure** - Following the correct file and folder structure.
3. **Frontmatter** - Getting the frontmatter right for discoverability.
4. **Templates** - Using the templates to speed up creation.
