# Planner-Architect Examples

This directory contains purpose-specific examples for incremental planning and execution patterns.

## Available Examples

- **[planning-patterns.md](planning-patterns.md)** - Monolithic vs incremental planning approaches
- **[validation-examples.md](validation-examples.md)** - Task validation checkpoint templates
- **[execution-templates.md](execution-templates.md)** - Output templates for task completion
- **[failure-scenarios.md](failure-scenarios.md)** - Recovery patterns when planning fails

## Usage

Each example file focuses on a specific planning scenario:
1. **Planning** - How to break down requests into atomic tasks
2. **Validation** - How to verify tasks before execution
3. **Execution** - What output to show during task completion
4. **Recovery** - How to handle failures and user feedback
